# Logistics On-Time Delivery Analysis

## Objective
Analyze delivery performance to identify delay drivers and recommend operational improvements.

## Dataset
Simulated e-commerce logistics dataset containing order details, courier assignments, locations, and delivery status.

## Tools Used
- Microsoft Excel (PivotTables, PivotCharts, Dashboards)

## KPIs
- On-Time Delivery Rate
- Number of Delayed Orders
- Courier On-Time Performance
- City-Level Performance

## Key Insights
- Identified couriers with lower on-time performance.
- Highlighted cities with higher delay rates.
- Found express deliveries perform better than standard deliveries.

## Recommendations
- Improve SLA monitoring for underperforming couriers.
- Reallocate delivery capacity during peak periods.
- Use historical data for demand forecasting.
